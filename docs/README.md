## 기능목록

> ### 요구사항

#### 컴퓨터
 - 1에서 9까지 `서로 다른 임의의 수 3개`를 선택

#### 사용자
- `서로 다른 3개의 숫자`를 입력

#### 컴퓨터 기능
- 입력한 숫자에 대한 결과 출력
  - 같은 수가 같은 자리에 있으면 `스트라이크`
  - 다른 자리에 있으면 `볼`
  - 같은 수가 전혀 없으면 `낫싱`
- 입력한 숫자와 컴퓨터가 생성한 숫자가 모두 맞으면 게임이 종료
  - 1 : 게임 재시작
  - 2 : 게임 종료

> ### 예외
- 사용자 입력
  - [ ] 사용자가 아무값도 입력하지 않는 경우
  - [ ] 사용자가 int 범위를 벗어나는 값을 입력한 경우
  - [ ] 입력 형식(ex.123, 345)이 잘못된 경우
  - [ ] 숫자를 입력하지 않는 경우
  - [ ] 중복되는 값을 입력하는 경우

- 게임 종료 후
  - [ ] 사용자가 아무값도 입력하지 않는 경우
  - [ ] 사용자가 int 범위를 벗어나는 값을 입력한 경우
  - [ ] 숫자를 입력하지 않는 경우
  - [ ] 1이나 2 외에 다른 값을 입력하는 경우
